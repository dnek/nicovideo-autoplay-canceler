# nicovideo-autoplay-canceler
[ニコニコ動画](https://www.nicovideo.jp/video_top)で動画の自動再生が開始された直後に自動的に一時停止します。
停止に少しラグがあるため、停止時の再生時間が3秒未満の場合は0秒まで戻します。
